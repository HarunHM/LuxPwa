INTRODUCTION TO R language.

R is a programming language and software environment for statistical analysis, graphical representation and reporting. It was created by Ross Ihaka and Robert Gentleman, hence naming the language R from the first letter of Developers, and is currently developed by the R Development Core Team. R is an Open source statistical programming language under GNU General Public License and is available for al operating systems, that is Linux, Windows and Mac.
